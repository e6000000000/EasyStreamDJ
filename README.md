# EasyStreamDJ
Скрипт позволяющий быстро добавлять треки на streamdj.ru

## Установка
1. скачать [python](https://www.python.org/downloads/)
2. установить requests `pip install requests`

## Использование
1. добавить в links.json интересующие вас треки (нужно только для второго и третьего режима работы)
2. запустить ezStreamDj.py
3. выбрать нужный вам режим работы
4. приступить к использованию
